# Hyprland — Руководство пользователя

## Первый запуск

После входа через `greetd` (экран логина) вы попадёте в Hyprland. Экран может быть пустым — это нормально.

1. Нажмите **`Super + Enter`** — откроется терминал `kitty`
2. Нажмите **`Super + R`** — откроется меню запуска приложений `rofi`
3. Начните работать!

| Обозначение | Клавиша |
|---|---|
| `Super` | Клавиша Windows (⌘ на Mac) |
| `Shift` | Shift |
| `Ctrl` | Ctrl |
| `Alt` | Alt |

## Основные хоткеи

### Запуск и управление окнами

| Хоткей | Действие |
|---|---|
| `Super + Enter` | Открыть терминал (kitty) |
| `Super + R` | Меню запуска приложений (rofi) |
| `Super + Q` | Закрыть активное окно |
| `Super + F` | Полноэкранный режим |
| `Super + V` | Переключить плавающий/тайлинговый режим |
| `Super + M` | Выйти из Hyprland (logout) |
| `Super + P` | Псевдо-тайлинг (разделить окно) |
| `Super + J` | Переключить разделение (горизонтальное/вертикальное) |

### Быстрый запуск приложений

| Хоткей | Приложение |
|---|---|
| `Super + B` | Google Chrome |
| `Super + E` | Nautilus (файловый менеджер) |
| `Super + C` | VS Code |

## Навигация

### Перемещение фокуса (vim-like)

| Хоткей | Направление |
|---|---|
| `Super + H` | Влево ← |
| `Super + L` | Вправо → |
| `Super + K` | Вверх ↑ |
| `Super + J` | Вниз ↓ |

### Перемещение окон стрелками

| Хоткей | Действие |
|---|---|
| `Super + ←/→/↑/↓` | Передвинуть окно в направлении |

### Ресайз окон

| Хоткей | Действие |
|---|---|
| `Super + Ctrl + ←` | Уменьшить ширину |
| `Super + Ctrl + →` | Увеличить ширину |
| `Super + Ctrl + ↑` | Уменьшить высоту |
| `Super + Ctrl + ↓` | Увеличить высоту |

> 💡 Также можно зажать `Super` и тянуть окно мышью или правой кнопкой для ресайза.

## Рабочие столы (Workspaces)

Всего доступно **10 рабочих столов**.

| Хоткей | Действие |
|---|---|
| `Super + 1–0` | Переключиться на стол 1–10 |
| `Super + Shift + 1–0` | Переместить окно на стол 1–10 |
| `Super + колёсико мыши` | Пролистать рабочие столы |

### Жесты тачпада

| Жест | Действие |
|---|---|
| Свайп 3 пальцами влево/вправо | Переключить рабочий стол |

## Медиа-клавиши

### Громкость

| Клавиша | Действие |
|---|---|
| `Fn + громкость+` | Увеличить громкость на 5% |
| `Fn + громкость-` | Уменьшить громкость на 5% |
| `Fn + mute` | Выключить/включить звук |
| `Fn + mic mute` | Выключить/включить микрофон |

> 💡 Для графического управления звуком запустите `pavucontrol`.

### Яркость экрана

| Клавиша | Действие |
|---|---|
| `Fn + яркость+` | Увеличить яркость на 5% |
| `Fn + яркость-` | Уменьшить яркость на 5% |

## Автозапуск

При старте Hyprland автоматически запускаются:

- **waybar** — панель статуса (верхняя панель)
- **dunst** — уведомления
- **cliphist** — история буфера обмена
- **blueman-applet** — Bluetooth в трее
- **nm-applet** — Network Manager в трее

---

## Плавающие окна

Следующие приложения автоматически открываются в плавающем режиме:

- Polkit (запрос пароля для sudo)
- Pavucontrol (управление звуком)
- Network Manager
- Bluetooth Manager
- Nautilus (файлы)
- Picture-in-Picture в браузерах

## Изменение конфигурации

Конфигурация Hyprland хранится в репозитории:

```
flakes/hyprland.conf
```

### Порядок изменения:

1. Отредактируйте `flakes/hyprland.conf` на Mac
2. Запушьте на GitHub
3. На ноутбуке:
   ```bash
   cd ~/Documents/github/nixos-laptop
   git pull
   sudo nixos-rebuild switch --flake .#honor
   ```
4. Конфиг автоматически обновится в `~/.config/hypr/hyprland.conf`

### Перезагрузка конфига без пересборки:

```bash
# Если вы изменили конфиг напрямую на ноутбуке:
hyprctl reload
```

> ⚠️ При следующем `nixos-rebuild` ваши локальные изменения будут перезаписаны конфигом из репозитория.

## Устранение неполадок

### Пустой экран после входа
```bash
# Проверьте, что Hyprland запущен
hyprctl version

# Проверьте конфиг на ошибки
hyprctl reload
```

### Не работает Waybar
```bash
# Перезапустить waybar
killall waybar && waybar &
```

### Не работают уведомления
```bash
# Перезапустить dunst
killall dunst && dunst &
```

### Не переключается раскладка
```bash
# Проверить текущую раскладку
hyprctl getoption input:kb_layout
hyprctl getoption input:kb_options
```

### Тачпад не работает как ожидается
Проверьте настройки в конфиге:
```bash
grep -A 5 "touchpad" ~/.config/hypr/hyprland.conf
```

## Полезные команды

```bash
# Информация о текущем состоянии
hyprctl monitors        # Мониторы
hyprctl clients         # Открытые окна
hyprctl workspaces      # Рабочие столы
hyprctl devices         # Устройства ввода
hyprctl version         # Версия Hyprland

# Управление
hyprctl dispatch exit   # Выйти из Hyprland
hyprctl reload          # Перезагрузить конфиг
```
